## Company-Specific Notes

**Amazon DE:** Hardest OA — 11-section Mettl MCQ (100 min) covering SQL, Python, DBMS, CS Networks, Linux, Data Viz, Pseudocode + Work Simulation (~50 min behavioral, silent killer regardless of coding score). Role-specific MCQ sections weighted heavily.

**Flipkart DE:** Machine coding (Spark + JSON processing) is the real differentiator in Round 2. Spark internals depth (Tungsten, Catalyst, broadcast joins) heavily weighted in Round 3.

**Walmart DE:** Most demanding pipeline (5–6 rounds). Spark optimization (delta lake, broadcast joins, CBO) + Cloud (AWS boto3) + System Design (event-driven) + Java (late round) all tested.

**PhonePe:** Pure DSA funnel — 3 OA coding problems, 2 more tech rounds with harder DSA + OS/Linux/DP. No SQL or ML in early rounds.

**Razorpay:** Machine coding focus — build an in-memory DB or complex system from scratch. Backend system design (pub-sub, load balancers) heavily weighted.

**Juspay:** Uniquely graph/tree-heavy OA, plus a multi-hour hackathon with thread safety + optimization. CS fundamentals (DBMS, OS, CN, OOP) in later rounds.

**ZS Associates (GenAI):** LLM benchmark definitions (MMLU, HELM, HumanEval, Big-Bench) asked alongside DSA coding in Round 2. RAG + VectorDB + LLM hyperparameters in Round 1.
