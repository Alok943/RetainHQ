## Interview Funnel Summary (per role)

### SDE
- **ROUND-1 ELIMINATOR:** DSA (coding OA), Aptitude MCQ, Core CS MCQ (OS/DBMS/Networks), Programming language gate
- **INTERVIEW TOPIC:** System Design (Round 2, rejection risk), SQL (final F2F), OOP
- **ON-THE-JOB:** Cloud/AWS, Git
- **Not interviewed for freshers:** Linux, Docker, LLD (gap — likely Optional), REST/API (gap)

### Backend Engineer
- **ROUND-1 ELIMINATOR:** Aptitude + Java + Core CS (OS/DBMS/Networks) bundled in MCQ OA
- **INTERVIEW TOPIC:** DSA (tech rounds 3–4 at mid-tier), REST/API, SQL, System Design (distributed)
- **ON-THE-JOB:** Docker, Cloud (low signal)
- **Not interviewed:** BI Tools, ML/DL, Statistics

### Data Analyst
- **ROUND-1 ELIMINATOR:** SQL (beginner–intermediate), Aptitude/Logical Reasoning
- **INTERVIEW TOPIC:** Advanced SQL (window functions, CTEs, query optimization), Statistics (A/B testing basics), ML fundamentals awareness, Gen AI awareness
- **ON-THE-JOB:** Power BI, Tableau, Excel, Python
- **Not interviewed:** DSA, Java, Docker, System Design

### Data Scientist
- **ROUND-1 ELIMINATOR:** ML Fundamentals (regression/classification), Statistics (probability, hypothesis testing), SQL basics
- **INTERVIEW TOPIC:** Deep Learning (advanced), NLP (Hugging Face, BLEU), System Design (ML system design), Advanced Stats (Markov, L2 norm)
- **ON-THE-JOB:** Python, BI Tools, Excel, LLMs (emerging)
- **Not interviewed:** DSA, Java, Docker, MLOps, Big Data
